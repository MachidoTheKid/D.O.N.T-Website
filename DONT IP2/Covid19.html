Learn About COVID-19

The COVID-19 API is a free to access data source created by Kyle Redelinghuys. It currently collates all information on the current state of the COVID-19 Pandemic across each country across the globe.

All data collected by the COVID-19 API is sourced from John Hopkins CSSE. Currently, although the API is free to use, users are limited to a certain amount of calls per day. Exceeding the call limit will result in the user being rate limited. In other words, users won't be able to make any more requests to the api for a certain amount of time. In order to avoid this, it is possible to subscribe to the API service. Each subscription package comes with certain perks. The most basic perk is the removal of the rate limit.

How It Works

Acquire The COVID-19 Data

As mentioned before, all data collected by the API is sourced from John Hopkins CSSE. At present, the API allows the user to organise the data under a certain number of formats. Data on COVID-19 is mainly divided up according to country. After that, the user can use specific JSON files to collect specific data on each country. For instance, the user can acquire data concerning infections on the first day of the pandemic, from day 1. The user can also get a general summary of infections, panning across all countries.

Initialise The Chart

In order to visualise the data, we first need something to visualise it with. An easy to use Javascript charting tool is Charts.js. It's a simple to use, lightweight solution that developers can take advantage of in order to create good looking charts with custom data. The chart can then be further customised, as there are many different available chart formats to choose from. In order to represent the COVID-19 data, a simple chart to use would be a bar chart.

Push Data To The Chart

All relevant COVID-19 Data is then pushed to a graph that the user can then interact with. The graph currently holds information on the total number of new infections found, the total number of COVID-19 related deaths, the total number of infections and so on. The chart is a bar graph that is constantly kept up to date thanks to the data provided by the COVID-19 API.
