Learn About COVID-19

The COVID-19 API is a free to access data source created by Kyle Redelinghuys. It currently collates all information on the current state of the COVID-19 Pandemic across each country across the globe.

All data collected by the COVID-19 API is sourced from John Hopkins CSSE. Currently, although the API is free to use, users are limited to a certain amount of calls per day. Exceeding the call limit will result in the user being rate limited. In other words, users won't be able to make any more requests to the api for a certain amount of time. In order to avoid this, it is possible to subscribe to the API service. Each subscription package comes with certain perks. The most basic perk is the removal of the rate limit.

How It Works

Acquire The Data

All the data for the earthquakes is gathered from the USGS (United States Geological Survey) website. The API allows the user to gather data on earthquakes based on time and magnitude. Links are provided to each GeoJSON file, which are each dedicated to a specific minimum magnitude and timeline.

Initialise The Map

The data is visualised with the help of a Google Map provided by the Google Maps API. The map API pulls on the data provided by the USGS API. Using the Google Maps API, developers can create their very own Google Map and customise it to their will.

Push Data To The Map

The data provided by the USGS API can be visualised with the help of the Google Maps API. The map's markers can then be customized in order to reflect the clae and magnitude of the earthquakes.
