About GeoJSON

What Is GeoJSON?

GeoJSON is used in order to represent geographical data, based on JSON.

Typically, it can be used to visualise a variety of different elements. Namely, a region of space (a Geometry), a spatially bounded entity (a Feature), or a list of Features (a FeatureCollection). GeoJSON can be used in conjunction with multiple Geometry types such as: Point, LineString, Polygon, MultiPoint, etc... Features in GeoJSON contain a Geometry object and additional properties, and a FeatureCollection contains a list of Features.

What Makes Up GeoJSON

Geometry Object

This item represents various points, curves and lines that are found in coordinate space. In layman's terms, it is essentially the location information. A Geometry Object's "type" member will always be one of the seven geometry types.

Feature Object

A Feature Obect is essentially something that is bound by space. Feature Objects are ALWAYS GeoJSON Objects, irrespective of where they may appear in the GeoJSON file. Feature Objects are typically associated with random data.

Feature Collection

A FeatureCollection object has a member with the name “features”. The value of “features” is a JSON array. Each element of the array is a Feature object as defined above. It is possible for this array to be empty.
