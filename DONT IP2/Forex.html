Forex Crypto Stock

The Forex Crypto Stock API is an API that provides real-time exchange rate prices for 145+ world currencies. The API comes with multiple endpoints, each serving a different response. Each response includes the latest exchange rate prices for all currencies, historical data of single or multiple currencies and the ability to convert the price from one currency to another.

They have an extensive documentation on how to access the API data through the different endpoints, most of which are accessible on the basic free plan. A restriction is set however to a maximum of 500 calls a month. this was sufficient for the sake of this project, but the limit can be pushed significantly to 10,000 calls a month on the cheapest plan, which costs $8 a month.

How It Works

Acquire The Data

All the Forex data is gathered from the FCS (Forex Crypto Stock) website. The API allows the user to recieve responses with all the latest forex values for each currency, but the search can be refined based on a number of specifiable parameters such as the base curency, the period between which the data should be spaced, the duration over which the user requires data on and many more.

Initialise The chart

The data is visualised onto a HTML canvas through Javascript with the help of the Charts.js framework. the chart pulls the filtered data received from the API and plots the values accordingly.

Visualize the data onto the graph

The data is presented in the form of a line graph displaying the opening, highest, lowest and closing values of the searched country's currency against the US dolar respectively. Each data point is presented in its own line for ease of visibility.
